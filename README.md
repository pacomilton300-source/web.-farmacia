 Sis457Farmacia

Descripción del negocio

Sis457Farmacia es un sistema de escritorio que permitirá gestionar una farmacia, controlando
productos, clientes y ventas de medicamentos. El sistema ayudará a mejorar el control de inventario y evitar errores en las ventas.
